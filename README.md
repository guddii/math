# Math
A c++ project for two dimensional vectors and matrices

[![Build Status](https://travis-ci.org/guddii/math.svg?branch=master)](https://travis-ci.org/guddii/math)

## Requirements

- Python v2.3 or newer
- CMake v2.6.4 or newer
- Clang v3.4 or newer

## Build steps
```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
$ ctest -VV
```
